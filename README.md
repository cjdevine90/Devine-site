# Devine Site — Next.js (App Router)

## Run locally
npm install
npm run dev
# open http://localhost:3000

## Deploy to Vercel
- vercel.com → Add New → Project → Upload → select this folder (.zip)
- Defaults are fine.

## Swap placeholders
Replace images in `/public/assets/...` with your real files.

## Formspree Setup
1) Create a form at https://formspree.io
2) Copy your endpoint (e.g., https://formspree.io/f/abcd1234)
3) Replace the `action` URL in `app/page.tsx` with your endpoint.
